## Updating the `~/.hgrc` and `~/.hgignore` files

- Update the required files with either these or check the Phiction Docs for the default ones

### Steps for `~/.hgrc`

- Open the file in a editor or use terminal `nano command`

```bash
nano ~/.hgrc
```

- Inside the editor paste these config and change the username part

```bash
[ui]
# You'll need to use your instahyre.com username
username = Anirban Basak <anirban.basak@instahyre.com>
ssh = ssh -C
ignore = ~/.hgignore
annotate = windsurf

[defaults]
# Needed for Phabricator to get enough context.
diff = --unified 8

[extensions]
churn =
color =
hgext.extdiff =
hgext.purge =
hgext.rebase =
histedit =
mq =
pager =
shelve =
graphlog =
journal =
strip =

[extdiff]
cmd.kdiff3 =
cmd.icdiff=icdiff
opts.icdiff=--recursive -N -H --cols 192

[merge-tools]
meld.executable = meld

[merge-patterns]
** = meld

[phases]
# Needed for rebase to work properly.
publish = False

[pager]
pager = LESS='FSRX' less

[alias]

log.summary=bold
log.bookmark=cyan bold
log.tag=blue bold
log.user=magenta
changeset.public=green
changeset.draft=red bold
```

- Press `Ctrl + O and hit enter` to write in the file and then press `Ctrl + X` to exit the editor window

## Create a SSH Key pair and add it in phabricator

- To create a keypair use this command below

```bash
ssh-keygen -t rsa -b 4096 -C "anirban.basak@instahyre.com"
```

- To verify that the keygen is created write this

```bash
ls ~/.ssh/
```

- Print the contents inside the pubilc SSH Key and copy that.

```bash
cat ~/.ssh/id_rsa.pub
```

### Now to paste in the phabricator website follow this

- Open [Phabricator Account Settings](https://phab.instahyre.com/settings/)
- Click on “Personal Account Settings”
- Click “SSH Public Keys” on sidebar
- Click “SSH Key Actions” and upload your public key with your name (if you copy-paste then sometimes a space gets added automatically - ensure that no spaces get added before pasting)

## After completing devel installation you also need Arcanist

- We have our own Arcanist Repository and we are going to set it up first
- You can refer to these two docs for reference [Arcanist Quick Start](https://secure.phabricator.com/book/phabricator/article/arcanist_quick_start/#:~:text=Configure%20Your%20Project) and [Arcanist Setup](https://phab.instahyre.com/w/arcanist_setup/)

### Prerequisites for installing and using Arcanist

- Install php and its required dependencies for Arcanist

```bash
paru -S php81-cli php81-curl
```

### After its done then clone the Arcanist repository

- Clone the Arcanist Repository

```bash
hg clone ssh://hg@phab.instahyre.com:2023/diffusion/32/arcanist/
```

- Add  Arcanist to the Fish path

```bash
fish_add_path /path/to/arcanist/bin
exec fish
```

- Verify arc works

```bash
which arc
arc help
```

- Open your devel `root` folder and run this Arcanist command to setup Arcanist for devel project. This will show some instructions, simply follow them and you are good to go

```bash
arc install-certificate
```

- After you run and provide your `API Token`. Please open the `~/.arcrc` file using `nano` and check. It should look something like this

```json
{
  "hosts": {
    "https://phab.instahyre.com/api/": {
      "token": "<your-conduit-api-token>"
    }
  },
  "config": {
    "hosts": {
      "https://phab.instahyre.com/api/": {
        "token": "<your-conduit-api-token>"
      }
    },
    "default": "https://phab.instahyre.com/"
  }
}
```
