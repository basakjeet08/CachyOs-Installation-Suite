# Installing Dependencies and starting the project

### 1. Install the required system dependencies first

```bash
sudo pacman -S --needed base-devel openssl bzip2 readline sqlite xz tk libffi ncurses mariadb-libs libxml2 libxslt libjpeg-turbo libpng freetype2 pcre2
```

### 2. Open the repository and start the virtual environment

- Make sure to cd inside the `devel repository` and then `activate the pyenv environment` before moving forward in this.

```bash
cd devel
```

```bash
pyenv activate blob
```

### 3. Install the required pip dependencies

```bash
pip install pip==23.3.2 setuptools==67.8.0 wheel==0.42.0
```

### 5. Now pip install the project dependencies

```bash
pip install -r requirements/pip3.txt \
    --find-links https://pip.instahyre.com/wheels/py3/ \
    --trusted-host pip.instahyre.com

pip install -r requirements/pip3_no_deps.txt \
    --no-deps \
    --find-links https://pip.instahyre.com/wheels/py3/ \
    --trusted-host pip.instahyre.com

pip install selenium==4.10.0 pytz==2019.3 bs4 parsel 2captcha-python==1.2.2 faker
```

- After the installation is completed, please revert back the `pip3.txt` file back to normal

### 6. Check the installation

```bash
python -m pip check
```

### Updating the database config files

- Create/Update the `db_config.json` file under `devel/auction/ignore/db_config.json` and replace the contents with this.

```json
{
  "default": {
    "NAME": "auction",
    "USER": "<your_user_name>",
    "PASSWORD": "<your_password_if_any>",
    "HOST": "127.0.0.1"
  }
}
```

### Install the Linters in your system

- Follow this doc for installing the linters - https://phab.instahyre.com/w/auto_linters_setup_workflow/

### Now try running the project

- Head inside the devel/auction folder and then run the following command.

```bash
python manage.py runserver
```

### Additional Step

- If you get this error saying - "Host '172.18.0.1' is not allowed to connect to this MySQL server", then follow the below steps.

```bash
# Open the mysql container inside docker
docker exec -it mysql mysql -u root
```

- Inside the mysql shell run these commands

```sql
CREATE USER 'root'@'%' IDENTIFIED BY 'root';
GRANT ALL PRIVILEGES ON *.* TO 'root'@'%' WITH GRANT OPTION;
FLUSH PRIVILEGES;
```
