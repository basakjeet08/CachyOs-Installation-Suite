# Virtual Environment Setup

## Install Python
- First check if you have a working version of Python since most linux distributions come with Python pre-installed.
 
- To check you can use this command and it should show the python version installed.

```bash
python --version
python3 --version
```

Move ahead if you have `python 3` installed.

## Install pyenv

- Install pyenv

```bash
curl -fsSL https://pyenv.run | bash
```

- Step 3 : Add pyenv to your PATH.

```bash
fish_add_path ~/.pyenv/bin
```

- Step 4: Add pyenv initialization inside fish config - `~/.config/fish/config.fish`

```bash
    # Pyenv initialization
    if command -q pyenv
        pyenv init - | source
    end
```

- Step 5: Reload the shell and check if `pyenv` command works

```bash
exec fish
pyenv --version
```

- Step 6: Check the list of dependencies pyenv provides and then install the correct python version. We use `python 3.7.17`

```bash
pyenv install -list
pyenv install -list | grep 3.7.17
```

```bash
pyenv install 3.7.17    
```

- Step 7: Create a virtual environment. Replace `your_virtual_environment_name` with the desired name you want for the virtual environment

```bash
pyenv virtualenv 3.7.17 <your_virtual_environment_name>
```

- Step 9: Whenever you start working on the project activate the virtual environment

```bash
pyenv activate <your_virtual_environment_name>
```

- Step 10: Check the python version inside the virtual environment

```bash
python --version
```
